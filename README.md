# 🎵 Music Player (HTML Only)

## 📌 Overview

This is a simple music player web page built using only HTML. It uses the HTML `<audio>` element to play multiple audio tracks directly in the browser.

## 🚀 Live Demo

👉https://sudha2005.github.io/music-player-html/

## 🛠️ Built With

* HTML5

## 🎧 Features

* Play multiple audio tracks
* Built-in browser audio controls
* Loop functionality for songs
* Simple structured layout

